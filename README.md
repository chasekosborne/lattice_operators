# Operators

This repository contains a set of python modules for constructing Hadron operators in Lattice QCD and analyzing their transformation properties.

# Requirements

- `numpy>=1.24`
- `sortedcontainers>=2.4.0`
- `sympy>=1.12`

Install with:

```bash
pip install -r requirements.txt
```

# Docs

All documentation is contained within the [Github Wiki](https://github.com/chasekosborne/lattice_operators/wiki), alongside user guides to get started using this package.
