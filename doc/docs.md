# lattice_operators

This package handles the construction of hadron and two-hadron operators of definite momentum. Much of the lattice operators package is code adapted from the [following paper](https://journals.aps.org/prd/abstract/10.1103/PhysRevD.88.014511) by Colin Morningstar et al. in hopes to make operator construction simple and straightforward.

This page contains links to comprehensive guides on getting started, alongisde dictionary-style references to classes and their properties
